# securityonion-misp
- Clone the repo:   
`git clone https://github.com/weslambert/securityonion-misp`   
- Run the setup script:   
`sudo securityonion-misp/setup-misp`   
- Update rules (if desired):   
`sudo rule-update`   
- Confirm rules in place:    
`grep -i misp /etc/nsm/rules/downloaded.rules`
